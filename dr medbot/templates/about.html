What Dr Medbot is, and what it does?

Dr Medbot is an AI powered chatbot developed using python, flask and machine learning libraries and algorithms like nltk, tensorflow, tflearn, bag of words etc. It aims to provide trustable information by fetching the queries sent by users.

Tech Stack Used

  • HTML
  • CSS
  • Python
  • Flask

Libraries/Modules Used

  • Numpy
  • TensorFlow
  • NLTK
  • Tflearn
  • Beautiful Soup
  • Request
